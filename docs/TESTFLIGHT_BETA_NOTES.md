# SafarMate TestFlight Beta Notes

## What to Test

Plan a trip, save it locally, reopen it offline, and try the main travel tools:

- Generate a one-day and multi-day itinerary.
- Save, rename, duplicate, delete, and reopen saved trips.
- Add Travel Details for a flight, accommodation, reservation, and provider contact.
- Try Qibla direction with location allowed and denied.
- Search nearby mosques, halal restaurants, public toilets, car rental, public transport, taxi services, attractions, and weather.
- Try English, Arabic, Bahasa Indonesia, and Bahasa Melayu.
- Share a trip, copy an itinerary, export a calendar file, and use Print / Save PDF.
- Enable, test, and disable prayer notifications.

## Known Limitations

- Live maps, weather, exchange rates, opening status, nearby-place searches, directions, and external websites require internet access.
- Saved trips and travel details are stored locally on the device and are not cloud-synced.
- Map and place information comes from contributed open-data sources and may be incomplete or outdated.
- Halal information is based on source data and is not guaranteed or certified by SafarMate.
- iPhone prayer notifications currently use the default iOS notification sound unless a licensed short sound is added in a later build.
